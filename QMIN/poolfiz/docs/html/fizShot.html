Executes a shot and stores an event list representing the outcome of the shot. You should only ever need one instance of a fizShot, but we have provided a copy constructor and assignment operator.

ShotStatus execute(fizTable& onTable, fizTableState& state, double a, double b, double theta, double phi, double V, bool verbose = false)
Executes a shot on the table.
Parameters:
tab - reference to the fizTable on which to execute the shot
state - reference to the fizTableState with which to execute the shot
a - right/left english in mm
b - top/bottom english in mm
theta - cue stick elevation in degrees
phi - cue stick azimuth angle (heading) in degrees
V - pre-impact cue stick velocity in m/s (max. 4.5 m/s)
verbose - set to true for console output of shot data (optional, default = false)

vector<fizEvent> getEventList()
Returns the time-sorted list of events that occured on the last shot, in the form of a Standard Template Library vector (array) of fizEvent objects.

bool save(string tag = "", bool autogen = true)
Saves a text representation of the shot to the working directory. If autogen is not supplied or is supplied and is true, the filename is automatically generated and contains the optional string tag. Otherwise if autogen is false the string tag itself is used for the filename.

bool load(string fname, fizTableState& state, double &a, double &b, double &theta, double &phi, double &V)
Loads a shot from the specified file.
Parameters:
fname - the filename, including extension, of the shot to load
state - reference to the initial table state to be populated from the file
a - reference to the right/left english to be populated from the file
b - reference to the top/bottom english to be populated from the file
theta - reference to the cue stick elevation to be populated from the file
phi - reference to the cue stick azimuth angle (heading) to be populated from the file
V - reference to the pre-impact cue stick velocity to be populated from the file
