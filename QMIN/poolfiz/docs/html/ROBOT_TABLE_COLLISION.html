This integer is used within the bitmask integer returned by fizTable::isPhysicallyPossible() (as well as the fizG versions of that class). It indicates that the supplied shot parameters will result in the robot striking the table; the actual integer returned by these methods is simply the OR'ed combinations of the all the possibilities from the list under the heading 'Variables' in the HTML documentation or listed below in the header file; you can AND the result returned by these methods with any of the possibilities from the list below to test if that particular possibility is true.
